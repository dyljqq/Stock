#股票


这个工程是从我的app中剥离出来的一块，主要有一下四个功能：

* 股票分时图
* 股票K线图
* 股票盘口
* 通过股票代码查找股票

数据来源：网易股票以及新浪股票的接口

有需要的同学可以下载来看下，欢迎大家一起交流。

ps：原先是想要从github上捡个现成的k线控件，后来发现跟自己的项目出入都太大，所以索性自己重新写了一个。
